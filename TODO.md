## Photos Needed

All photos should be:
- **Format**: JPG or WebP
- **Resolution**: 72 DPI minimum (save originals at higher res)
- **Max file size**: Under 500KB each for web performance
- **Lighting**: Natural lighting preferred
- **Privacy**: No residents in photos
- **Feel**: Warm and comfortable, avoid clinical appearance

### How to Add Photos

1. Resize/crop photos to recommended dimensions
2. Compress for web (aim for under 500KB each)
3. Save to `images/` folder with descriptive filenames
4. Update `index.html` to replace `images/placeholder.svg` with new filename
5. Test that images display correctly
